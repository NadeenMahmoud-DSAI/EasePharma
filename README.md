# EasePharma (E-Pharmacy Platform)

**EasePharma** is a web-based platform designed to bridge the gap between customers and pharmacies. It allows users to search, compare, and order medicines from nearby stores seamlessly. The system solves the common issue of "out-of-stock" medications by aggregating inventory from multiple pharmacies into a single, easy-to-use interface.

## Key Features

### For Customers
* **Smart Search:** Quickly find available drugs across multiple nearby pharmacies.
* **Price Comparison:** Compare prices to get the best deal.
* **Order Tracking:** Real-time status updates on orders (Processing -> Shipped -> Delivered).
* **Support Chat:** Direct communication channel for inquiries.

### For Admins
* **Dashboard Analytics:** Overview of system performance and order volume.
* **Inventory Management:** Track stock levels across different pharmacy branches.
* **Support Inbox:** Centralized hub to manage and respond to customer tickets.

## Tech Stack
* **Core Framework:** Flask (Python)
* **Data Processing:** Pandas
* **Utilities:** Werkzeug
* **Containerization:** Docker
* **Frontend:** HTML/CSS (Jinja2 Templates)
